# Proyecto de Clase: Calculadora Básica

Este repositorio contiene el desarrollo de una calculadora sencilla, realizada como parte de mis prácticas en la materia de Programación I.

## Descripción
La calculadora permite realizar operaciones matemáticas básicas como:  
- Suma + 
- Resta -  
- Multiplicación *
- División /

El proyecto fue desarrollado para reforzar conceptos de lógica de programación y práctica en el uso de estructuras básicas.
 

## 🛠️ Tecnologías utilizadas
- Lenguaje: Java  
- Editor: Visual Studio Code
